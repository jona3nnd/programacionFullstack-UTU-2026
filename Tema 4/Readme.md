# 🌴 Vacaciones de Jonathan López

Página web creada como tarea de programación para presentar las actividades realizadas durante las vacaciones.

## Estructura

- `index.html` — estructura semántica con header, main (3 secciones) y footer
- `styles.css` — estilos organizados en Nivel 1 y Nivel 2
- `img/` — carpeta con las imágenes (descarga.jpg, descarga (1).jpg, descarga (2).jpg)

## Secciones

| Sección | Tema |
|---|---|
| 🎣 Fui a Pescar | Pesca en el Puente Nuevo |
| 🔧 También Trabajé | Instalación de calefón y electricidad |
| 📺 Maratonee una Serie | Serie Gotham |

## Propiedades CSS nuevas aplicadas (Nivel 2)

1. `box-shadow` — sombra en tarjetas
2. `border-radius` — esquinas redondeadas
3. `transition` — animación suave en hover
4. `transform: scale()` — zoom en imágenes al pasar el mouse
5. `letter-spacing` — espaciado entre letras en títulos
6. `backdrop-filter` — efecto vidrio esmerilado en el header
